# Sportify Store - Azzrak Project

A modern and professional Frontend E-commerce project for a sports store, featuring a sleek design that blends Deep Blue and Orange accents. The project focus is on delivering a premium User Experience (UX) and full responsiveness across all devices.

## âœ¨ Key Features

### 1. Dynamic Navbar
- **Countdown Timer**: Displayed at the top to highlight limited-time discounts.
- **Mobile Navigation**: Includes a custom-styled toggle menu optimized for mobile users.
- **Interactive Icons**: Search, User Profile, Wishlist, and a Cart with a live item counter.

### 2. Premium Hero Section
- **Custom Slider**: Showcases main store highlights with smooth transition effects.
- **Numerical Indicators**: A unique navigation system (01, 02, 03) allowing users to switch between featured slides easily.

### 3. Products Section
- **Hover Effects**: Products feature a dual-image system; hovering reveals a second angle or detail.
- **Quick Actions**: "Add to Cart" and "View Product" buttons are accessible directly from the product card.
- **Rating System**: Elegant star-based rating display for each product.
- **Promotion Banners**: An integrated grid of stylish marketing banners mixed with product listings.

### 4. Offers Slider
- **Interactive Carousel**: Showcases special deals with a focus-zoom effect on the active central slide.
- **Dual-Image Cards**: Each slide features a main product image paired with a secondary detail box for a high-end feel.

### 5. Advanced Footer
- **Fully Responsive**: Specially optimized for mobile, rearranging elements into a user-friendly vertical layout.
- **Newsletter**: Integrated subscription box for email marketing.
- **Social Media Integration**: Quick links to Instagram, Twitter, Snapchat, and Facebook.
- **Trust & Security**: Displays "Mawthooq" business verification, Tax ID, and multiple payment methods (Visa, MasterCard, Mada, PayPal).
- **Instagram Feed**: A curated mini-gallery showing the latest posts from the store's Instagram account.
